Tujuan Project
Tujuan dari pembuatan website ini adalah untuk melatih mahasiswa dalam membangun
website portofolio profesional, memahami penggunaan teknologi open source dalam
pengembangan web, serta membiasakan penggunaan GitHub sebagai version control.
Selain itu, website ini juga berfungsi sebagai portfolio digital mahasiswa.

Tech Stack
- HTML5 (Semantic Markup)
- CSS3 (Flexbox, Grid, Responsive Design)
- Vanilla JavaScript (ES6+)
- Git & GitHub
- GitHub Pages
- Google Fonts

Fitur Utama
Website ini memiliki beberapa fitur utama sesuai dengan ketentuan tugas, antara lain:
- Dark / Light Mode Toggle menggunakan CSS variables dan JavaScript, dengan
  preferensi mode yang tersimpan di localStorage
- Interactive Project Gallery dengan grid layout responsif, efek hover, filter
  kategori, serta modal detail project tanpa library eksternal
- Navigasi responsif dengan smooth scroll, active navigation indicator, dan
  hamburger menu untuk tampilan mobile
- Halaman Blog yang menampilkan artikel dengan judul, tanggal, ringkasan,
  dan kategori
- Contact Form dengan validasi client-side, pesan error yang informatif, serta
  success message setelah form berhasil dikirim
